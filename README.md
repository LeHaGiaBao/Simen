## Installing

Clone the repository using following command or download

```
git clone https://github.com/LeHaGiaBao/Simen.git
```

To install dependency

```
npm install
```

To start the server

```
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
